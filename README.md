# NSFW Guard AI Telegram Bot

AI-based Telegram group moderation bot.

## Features
- NSFW text detection
- Image, sticker, GIF & video scanning
- Auto delete NSFW content
- 3 warnings = auto mute
- Approved user bypass
- Admin controls

## Deployment
- Python 3.10
- Hosted on HiddenCloud
- Uses environment variable for BOT_TOKEN

## Security
⚠️ Do not put bot token in code or GitHub.
Use environment variables only.

## Commands
/approve user_id  
/unapprove user_id  
/scan_on  
/scan_off
